# dashboard-chamados-ti-powerbi
Dashboard em Power BI para análise de chamados de suporte técnico

 Dashboard de Desempenho de Chamados de TI

Projeto desenvolvido com foco em análise de dados de Service Desk, com o objetivo de identificar padrões de atendimento, monitorar indicadores de desempenho e apoiar melhorias operacionais.

## Objetivo

Analisar dados de chamados de suporte técnico para:

- Avaliar o desempenho do atendimento
- Monitorar o cumprimento de SLA
- Identificar gargalos nos processos
- Gerar insights para melhoria contínua


## Ferramentas Utilizadas

- Power BI
- Excel (estruturação de dados)
- Modelagem de dados básica

## Estrutura do Projeto

- `chamadosti.xlsx` → Base de dados utilizada na análise  
- `chamadosti.pbix` → Arquivo do Power BI com dashboard desenvolvido  

## Indicadores Analisados

- Total de chamados  
- Tempo médio de resolução  
- Percentual de SLA cumprido  
- Distribuição por categoria  
- Distribuição por prioridade  
- Chamados por técnico responsável  

## Visualizações do Dashboard

O dashboard permite visualizar:

- Volume de chamados ao longo do tempo  
- Comparação entre categorias (Hardware, Software, Rede)  
- Análise de prioridades (Alta, Média, Baixa) com semântica de cores  
- Distribuição de SLA (cumprido vs não cumprido)  
- Performance por técnico  

## Principais Insights

- Alta concentração de chamados na categoria de hardware  
- Chamados de alta prioridade apresentam maior impacto no SLA  
- Taxa de SLA em aproximadamente 65%, indicando oportunidade de melhoria  
- Diferença na distribuição de chamados entre técnicos
